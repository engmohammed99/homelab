*   **Docker:** This directory contains my projects related to Docker.

*   **FullStack:** A collection of projects covering various web technologies.
*   **Javascript:** A collection of my JavaScript projects.
*   **Kubernetes:** This directory contains my projects related to Kubernetes.
*   **Node js:** This directory contains my projects related to Node.js.

Each project is contained in its own directory. You can navigate to each directory to see the project's source code and any related files.
